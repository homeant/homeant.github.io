BeanFactoryиҜҰи§Ј

з”ұhomeant | 2019-04-22 14:03 жөҸи§Ҳ99ж¬Ў

BeanFactoryпјҢе®ғжҳҜдёҖдёӘжҺҘеҸЈе·ҘеҺӮзұ»пјҢз”ЁдәҺз®ЎзҗҶspringдёӯзҡ„BeanгҖӮеңЁSpringдёӯпјҢBeanFactoryжҳҜIOCе®№еҷЁзҡ„ж ёеҝғжҺҘеҸЈпјҢе®ғзҡ„иҒҢиҙЈеҢ…жӢ¬пјҡе®һдҫӢеҢ–гҖҒе®ҡдҪҚгҖҒй…ҚзҪ®еә”з”ЁзЁӢеәҸдёӯзҡ„еҜ№иұЎеҸҠе»әз«ӢиҝҷдәӣеҜ№иұЎй—ҙзҡ„дҫқиө–гҖӮ SpringдёәжҲ‘д»¬жҸҗдҫӣдәҶи®ёеӨҡжҳ“з”Ёзҡ„BeanFactoryе®һзҺ°пјҢXmlBeanFactoryе°ұжҳҜеёёз”Ёзҡ„дёҖдёӘпјҢиҜҘе®һзҺ°е°Ҷд»ҘXMLж–№ејҸжҸҸиҝ°з»„жҲҗеә”з”Ёзҡ„еҜ№иұЎеҸҠеҜ№иұЎй—ҙзҡ„дҫқиө–е…ізі»гҖӮXmlBeanFactoryзұ»е°ҶжҢҒжңүжӯӨXMLй…ҚзҪ®е…ғж•°жҚ®пјҢе№¶з”Ёе®ғжқҘжһ„е»әдёҖдёӘе®Ңе…ЁеҸҜй…ҚзҪ®зҡ„зі»з»ҹжҲ–еә”з”ЁгҖӮ

  • BeanFactoryжҺҘеҸЈ
    • жҳҜSpring beanе®№еҷЁзҡ„ж №жҺҘеҸЈ.жҸҗдҫӣиҺ·еҸ–bean,жҳҜеҗҰеҢ…еҗ«bean,жҳҜеҗҰеҚ•дҫӢдёҺеҺҹеһӢ,иҺ·еҸ–beanзұ»еһӢ,bean еҲ«еҗҚзҡ„ж–№жі•
  • BeanFactoryзҡ„дёүдёӘеӯҗжҺҘеҸЈ
    • HierarchicalBeanFactoryпјҡжҸҗдҫӣзҲ¶е®№еҷЁзҡ„и®ҝй—®еҠҹиғҪ
    • ListableBeanFactoryпјҡжҸҗдҫӣдәҶжү№йҮҸиҺ·еҸ–Beanзҡ„ж–№жі•
    • AutowireCapableBeanFactoryпјҡеңЁBeanFactoryеҹәзЎҖдёҠе®һзҺ°еҜ№е·ІеӯҳеңЁе®һдҫӢзҡ„з®ЎзҗҶ
  • ConfigurableBeanFactory
  • ConfigurableListableBeanFactory
    • з»§жүҝдәҶдёҠиҝ°зҡ„жүҖжңүжҺҘеҸЈпјҢеўһеҠ дәҶе…¶д»–еҠҹиғҪпјҡжҜ”еҰӮзұ»еҠ иҪҪеҷЁ,зұ»еһӢиҪ¬еҢ–,еұһжҖ§зј–иҫ‘еҷЁ,BeanPostProcessor,дҪңз”Ёеҹҹ,beanе®ҡд№ү,еӨ„зҗҶbeanдҫқиө–е…ізі», beanеҰӮдҪ•й”ҖжҜҒвҖҰ
  • е®һзҺ°зұ»DefaultListableBeanFactory
    • е®һзҺ°дәҶConfigurableListableBeanFactoryпјҢе®һзҺ°дёҠиҝ°BeanFactoryжүҖжңүеҠҹиғҪгҖӮе®ғиҝҳеҸҜд»ҘжіЁеҶҢBeanDefinition

ClassPathXmlApplicationContextжҳҜBeanFactoryзҡ„дёҖдёӘе…ёеһӢзҡ„е®һзҺ°пјҢеҗҢж—¶е®ғд№ҹжҳҜ ApplicationContextзҡ„еӯҗзұ»пјҢClassPathXmlApplicationContextд»Һ xml зҡ„й…ҚзҪ®ж–Үд»¶дёӯиҺ·еҸ– bean е№¶дё”з®ЎзҗҶд»–д»¬гҖӮ

public static void main(String[] args) throws Exception {
    BeanFactory bf = new ClassPathXmlApplicationContext("application.xml");
    UserHandler userHandler = (UserHandler) bf.getBean("userHandler");
    log.info("handler:{}",userHandler);
}

xmlй…ҚзҪ®

<bean id="userHandler" class="fun.vyse.cloud.UserHandler">
    <property name="userService" ref="userService"/>
</bean>
